The Annotator

The Annotator is a Javascript tool that can be added to any webpage to allow annotation. Annotations can have comments, tags, users and more. See a live demonstration or more in the developers section below.

The Bookmarklet

The Bookmarklet is a simple tool that allows you to add the annotator to any webpage and save your annotations to AnnotateIt. Check out the Bookmarklet demo or get your very own bookmarklet right now by signing up for an account.

Developers

Allowing your users to annotate a webpage is easy. First you'll need to download the source code and include it on your page along with jQuery. Then simply add the following lines to load the Annotator, allowing users with AnnotateIt accounts to start annotating immediately:

jQuery('#content').annotator()
                  .annotator('setupPlugins');

If you want more control, or you don't want your users (who might already have accounts on your website) to need accounts here at AnnotateIt, we provide support for delegated authentication: you control who gets to annotate, and AnnotateIt will store annotations for you and your users.

Setting up delegated authentication is a little more involved, and you should go see the documentation for the full details. That said, once you've set up your token generator (see the docs) the code is still pretty simple: 

jQuery('#content').annotator()
                  .annotator('setupPlugins', {
                      tokenUrl: 'http://example.com/getToken'
                  });
